Lancement des .jar exécutables :

- Lancement du serveur : java -jar ServerLaunch

- Lancement de l'interface client : java -jar ClientLaunch

How to :

- Pour s'inscrire, appuyer sur le bouton "Register", renseignez login et mot de passe et vous serez automatiquement connecté au chat.
- Pour vous connecter, appuyer sur le bouton "Connect", renseignez login et mot de passe et vous serez connecté au chat après vérification des informations.
- Pour envoyer des messages normaux, tapez votre message et appuyez sur Entrer ou sur le bouton "Send".
- Pour envoyer des messages privés, entrez tout d'abord "@[utilisateur] *espace*" et ensuite votre message, puis appuyez sur Entrer ou sur le bouton "Send".
